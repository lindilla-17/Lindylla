# Ways of Working — ¿Qué es esto?

> **En lenguaje normal:** Las "formas de trabajar" son las reglas del juego
> entre tú y Claude. Como un contrato de trabajo, pero para una IA.
> Definen quién decide qué, cómo se hacen los cambios, y qué está prohibido.

---

## Por qué existen estas reglas

Sin reglas claras, Claude puede:
- Tomar decisiones de negocio que le corresponden a ti
- Hacer cambios grandes y difíciles de deshacer
- Olvidar el contexto de sesiones anteriores
- Ser demasiado técnico y difícil de entender

Con estas reglas, Claude sabe exactamente cómo comportarse.
