# Memory — ¿Qué es esto?

> **En lenguaje normal:** Esta carpeta es la memoria de largo plazo de Claude.
> Aquí guarda lo que aprende sobre el proyecto, el equipo y las preferencias
> de trabajo para que no se pierda entre sesiones.

---

## ¿Por qué existe?

Claude no recuerda conversaciones anteriores. Cada sesión nueva empieza de cero.
Sin este sistema, tendríamos que explicarle lo mismo una y otra vez.

Con el sistema de memoria, Claude lee estos archivos al inicio de cada sesión
y recupera todo el contexto aprendido.

---

## Qué guarda Claude aquí

- **Preferencias del equipo** — formatos, estilos, formas de comunicarse
- **Contexto del proyecto** — decisiones informales, matices de negocio
- **Correcciones pasadas** — para no repetir el mismo error
- **Referencias útiles** — dónde está documentado tal cosa, quién sabe qué

---

## ¿Tengo que hacer algo?

No. Claude gestiona esta carpeta automáticamente.
Puedes pedirle que "recuerde" algo concreto:
*"Recuerda que preferimos los importes con coma decimal, no con punto"*

También puedes pedirle que "olvide" algo:
*"Olvida lo que tienes sobre el módulo de nóminas, ya no aplica"*
